# Changelog

All notable changes to the ForgeStack SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-12-03

### Added
- Initial release of ForgeStack SDK
- TypeScript client for ForgeStack API
- Support for Organizations API
  - List, create, get, update, delete organizations
- Support for Projects API
  - List, create, get, update, delete projects
- Support for API Keys API
  - List, create, get, update, revoke, rotate API keys
- Support for Dashboard API
  - Get dashboard summary
- Support for Health API
  - Health check endpoint
- Full TypeScript type definitions
- CommonJS and ESM module support
- Context management (organization, access token, API key)
- Error handling with custom error callback
- Comprehensive documentation and examples
