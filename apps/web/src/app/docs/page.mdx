# Introduction

Welcome to ForgeStack, a production-ready multi-tenant SaaS starter kit.

## What is ForgeStack?

ForgeStack is a complete foundation for building SaaS applications. It includes:

- **Authentication** - Email/password, OAuth, magic links
- **Multi-tenancy** - Organizations, teams, invitations
- **Billing** - Stripe subscriptions and usage-based billing
- **RBAC** - Fine-grained role-based access control
- **API Keys** - Secure API key management
- **Webhooks** - Outgoing webhooks with retry logic
- **File Uploads** - S3/R2 compatible storage
- **And more...**

## Tech Stack

- **Frontend**: Next.js 15, React 19, Tailwind CSS
- **Backend**: NestJS, PostgreSQL, Drizzle ORM
- **Auth**: better-auth
- **Payments**: Stripe
- **Queue**: BullMQ + Redis
