# Installation

Get ForgeStack up and running in minutes.

## Prerequisites

- Node.js 20+
- pnpm 9+
- Docker (for local development)
- PostgreSQL 15+
- Redis 7+

## Clone the Repository

```bash
git clone https://github.com/forgestack/forgestack.git
cd forgestack
```

## Install Dependencies

```bash
pnpm install
```

## Set Up Environment

```bash
cp .env.example .env
```

Edit `.env` with your configuration.

## Start Services

```bash
docker compose up -d
pnpm db:migrate
pnpm dev
```

<Callout type="success">
  Your app is now running at http://localhost:3000
</Callout>
