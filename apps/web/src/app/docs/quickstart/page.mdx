# Quickstart

Build your first feature in 5 minutes.

## Create Your First Organization

After signing up, you'll be guided through creating your first organization.

## Create a Project

Navigate to Projects and click "New Project".

## Generate an API Key

Go to Settings > API Keys and create a new key.

## Make Your First API Call

```bash
curl -X GET https://api.yourapp.com/api/v1/projects \
  -H "X-API-Key: your-api-key" \
  -H "X-Org-Id: your-org-id"
```

<Callout type="info">
  Check out the [API Reference](/docs/api/overview) for all available endpoints.
</Callout>
